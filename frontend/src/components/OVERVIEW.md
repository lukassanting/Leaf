# `frontend/src/components` Overview

This folder contains the Leaf frontend UI building blocks.

## Subfolders

- `editor/`: the rich editor surface and editor-specific widgets.
- `database/`: database table view UI (columns, rows, cell editing).
- `page/`: page-level UI primitives like identity headers and icon pickers.

## Top-level components

- `Sidebar` / `SidebarTree` and related components implement navigation and tree interactions.
- `StatusBar`, `NavigationProgress`, `AIAssistant`, etc. provide shared UI chrome.
