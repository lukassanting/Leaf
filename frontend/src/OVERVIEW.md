# `frontend/src` Overview

This folder contains the Leaf frontend codebase.

Main areas:

- `app/`: Next.js App Router route entrypoints (layouts + pages).
- `components/`: Reusable UI pieces (sidebar, editor, database table UI, etc.).
- `hooks/`: Client-side state and data-loading hooks.
- `lib/`: API client wrappers, cache/mutation helpers, and small utilities.
