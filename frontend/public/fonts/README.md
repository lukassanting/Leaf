# Campaign theme fonts (vendored)

These `.woff2` files are committed so `npm install` and Docker builds work **without** fetching from Google Fonts.

| File | Family |
|------|--------|
| `cinzel.woff2` | Cinzel |
| `cinzel-decorative-*.woff2` | Cinzel Decorative |
| `crimson-pro-*.woff2` | Crimson Pro |

Licensed under the [SIL Open Font License](https://scripts.sil.org/OFL) via [Google Fonts](https://fonts.google.com/).

To refresh from upstream (maintainers): from `frontend/`, run `npm run fonts`.
